CHAX API Test Commands

Copy/paste these into PowerShell to validate the server and endpoints.

Health

Invoke-RestMethod -Uri "http://127.0.0.1:8080/api/v1/health"

Echo

Invoke-RestMethod -Uri "http://127.0.0.1:8080/api/v1/echo" `
  -Method Post `
  -ContentType "application/json" `
  -Body '{"payload":"hello from PS"}'

List checklists

Invoke-RestMethod -Uri "http://127.0.0.1:8080/api/v1/checklists"

List slugs for a checklist

$checklist = "chax-demo"
Invoke-RestMethod -Uri "http://127.0.0.1:8080/api/v1/slugs?checklist=$($checklist)"

Update a slug

$id = "ADDRESS_ID_HERE"
Invoke-RestMethod -Uri "http://127.0.0.1:8080/api/v1/slugs/$id" `
  -Method Patch `
  -ContentType "application/json" `
  -Body '{"status":"Pass","result":"5.0 V","comment":"manual update"}'
