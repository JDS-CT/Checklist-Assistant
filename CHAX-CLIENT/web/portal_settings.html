Portal Settings

Configure the host and port for API calls. Values are stored in your browser and used by the checklist assistant.

Default: http://127.0.0.1:8080. If you change servers, update here first.

One entry per line. Use `name=path`; a selected folder may be either a `checklists` folder or a parent folder containing one. Checklists below it must use `checklists/<pack>/<checklist>/checklist.md`. Saved paths stay in `.chax/local_settings.json` on this PC.
